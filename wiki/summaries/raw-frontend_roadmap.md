## Key claims
- 前端开发从简单的 HTML/CSS/JS 演变到复杂的单页应用 (SPA) 和组件化架构。
- 现代前端开发必须掌握 ES6+ 的 JavaScript 核心特性，包括解构赋值、箭头函数、模板字符串、异步编程等。
- 现代前端框架如 React、Vue、Angular 均基于组件化和声明式渲染的思想。
- 构建工具如 Vite、npm、TypeScript 是现代前端开发工作流程不可或缺的组成部分。

## Methodology
本文通过列举关键技术和工具来描述前端开发学习路线与核心概念，没有使用具体的研究方法。

## Results
前端开发涵盖了核心基石如语义化 HTML 和 CSS 现代布局，现代 JavaScript 的特性和最佳实践。介绍了一系列流行的前端框架和库，以及这些框架如何通过组件化和声明式渲染提高开发效率。现代的构建工具如 Vite 提供了开发和部署代码的高效方式，同时 TypeScript 为复杂的前端项目提供了更好的类型检查和代码维护。

## Concepts
HTML5, CSS3, 语义化 HTML, Flexbox, Grid, CSS 变量, JavaScript (ES6+), 解构赋值, 箭头函数, 模板字符串, 异步编程, Event Loop, Promise, async/await, 模块化 (ES Modules), 现代前端框架, React, Vue, Angular, 组件化, 声明式渲染, JSX, Hooks, useState, useEffect, 虚拟 DOM, Vite, npm, pnpm, yarn, TypeScript
