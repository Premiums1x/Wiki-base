## Key claims
- � 罔络安全保护网络系统、硬件、软件和数据免遭受数字攻击、损坏或未经授权访问。
-   网络协议基于分层架构，经典参考模型为 TCP/IP 囂层模型或 OSI 七层模型。
-   Web 开发应防范常见的跨站脚本（XSS）、跨站请求伪造(CSRF)和 SQL 注入等常见安全威胁。
 -   密码学技术包括对称加密、非对结加密、HTTPS 协议和单向哈希函数用于数据安全。

## Method
描述了层架构在网络协议中的应用，以及防范常见安全威胁的技术的基础原理和防御方法。

## Results
-   TCP/IP 卽层模型包含有物理层、数据链路层、网络层和传输层，各自定义了数据传输的不同阶段。
 -   在 Web 开发展中，采用了防御措施如针对 XSS 和 CsRFF 的转义和启用 CSP、SQL 注入的使用参数化查询。
 -   密码学应用包括了对称加密算法如 AES、非对结加密应用密码如 RSA 和 ECC、以及 HTTPS 协议确保网络传输安全且单向哈希函数可用于密码存储以防止攻击。
